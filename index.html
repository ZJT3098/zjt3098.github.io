ZBittorrent 项目改动记录

1. 添加 SHA1 帮助类
代码中缺少 SHA1.HashData() 方法，添加了 SHA1Helper 类
改动原因：编译错误 - SHA1 类未定义
2. Bencode 支持 long 类型
Encode 方法添加 long 类型支持
改动原因：编译错误 - Bencode.Encode 不支持 long
3. 修复进度计算除零错误
在 percent 计算前添加 total > 0 检查
改动原因：运行时可能除零错误
4. 修复 UDP Tracker 端口转换
修复 port 字段的字节序转换，重命名变量避免冲突
改动原因：UDP Tracker 端口字段转换错误
5. 修复 TargetFramework
改为 net10.0-windows
改动原因：CsWinRT 生成错误
6. 修复 UDP 变量名冲突
重命名 port 变量为 peerPort/peerPortNum
改动原因：变量名在作用域中冲突
7. 修复分卷文件合并
添加文件存在检查和异常处理
改动原因：下载完后无法合并分卷文件
8. 修改为单文件写入
HttpDownloader 改为直接写入同一文件，用锁同步，多线程同时下载不同 Range
改动原因：创建多个分卷文件体验不佳
9. 添加设置页面
XAML 添加 TabControl 页面（下载、设置、日志）
改动原因：需要设置下载选项
10. 添加下载模式选择
单文件/多分卷 RadioButton
改动原因：选择下载方式
11. 添加线程数设置
Slider 1-32 线程
改动原因：设置下载线程数
12. 添加网速限制
TextBox 输入 KB/s，0=不限
改动原因：限制下载速度
13. 添加跳转连接
设置页面添加跳转按钮，使用系统默认浏览器打开
改动原因：快速打开 Tracker URL
14. 修复 nullable 警告
事件和属性添加 ? 允许 null
改动原因：编译警告
15. 代码设置初始值
构造函数设置 txtJumpUrl 默认值
改动原因：跳转链接需在代码中设置
